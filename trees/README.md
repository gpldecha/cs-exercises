# Binary Search Tree (BST)

* runway reservation system (airport has a single runway)


|		         | Average	| Worst |
| ---------- | :-------:| ----: |
| Space		   | O(n)	    | O(n)  |
| Search		 | O(log n) |	O(n)  |
| Insert		 | O(log n) |	O(n)  |
| Delete		 | O(log n) |	O(n)  |
